## üì¶ Installation
Install to the folder that needs to be created:
```
git clone https://github.com/OleksiiTarasenko/arnelify-pod-php
```

Create .env from .env.local:
```
cd arnelify-pod-php
cp .env.local .env
```

Run Docker:
```
docker compose up -d --build
```
